# Product

## Register
product

## Users & purpose
Freelancers, bloggers, e-commerce sellers, and privacy-conscious users who need quick image compression, resizing, or format conversion without uploading files to a cloud service. They arrive via search with a specific task in mind and want to complete it in under a minute.

## Primary workflow
Land on a tool page → drop image → adjust settings → download result. No account, no onboarding.

## Brand personality
Precise, calm, trustworthy. The interface should feel like a well-made native utility, not a marketing site.

## Anti-references
- Generic AI tool landing pages (purple gradients, glass cards, hero metrics)
- Over-decorated SaaS marketing sites
- Cream/beige "warm editorial" backgrounds
- Busy card grids with identical icon + title + description blocks

## Strategic principles
1. Privacy is the product — "no upload" must be visible but not shouty
2. Task-first — the drop zone is the hero on tool pages
3. Familiar affordances — standard buttons, forms, file inputs
4. Restrained color — accent only on primary actions and links

## Accessibility
WCAG AA minimum. Respect `prefers-reduced-motion`. All interactive elements need visible focus states.
